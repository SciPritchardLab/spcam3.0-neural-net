2. Using CAM2.0.2

2.4 Example Namelists

Several example namelist inputs are presented in the following sections. These examples were chosen to illustrate many of the namelist variables described in Sec. 2.2 . These examples assume a stand-alone configuration of CAM2.0.2. Furthermore, it is assumed that the root directory for the CAM2.0.2 dataset distribution is /data. (The root directory contains the subdirectories atm and lnd.)
